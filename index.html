Python Open Education Resource

Introduction

This website serves as a brief introduction to the Python Open Education Repository (OER) being developed by Ravi Patel & Dominic DiSanto within the University of Pittsburgh's School of Pharmacy. The materials referenced and linked to on this page are currently hosted publicily in a GitHub repository. Below we outline the materials included in this repository as a brief summary of the available resources! This project aims to serve as a reference of materials to learn and practice Python in the context of data science and/or machine learning in the pharmaceutical sciences. Our hope is that these lessons generalize beyond PharmacoAnalytics to help any user learn Python effectively! The materials contained within this repository target a diverse audience of Python users, including first-time Python users and those with previous experience.

Use Cases:

This repository contains a number of Use Cases, that is assignments that present users with data that represents a common and/or useful task they may face in their careers in academic research, quality assurance analysis, These uses cases are gleaned from previously published research and often in collaboration with academic and industry partners, who have shared their work as pharmacists and/or analysts to help us craft assignments that help users not only build their Python competencies (and confidence) but also gain experience in projects relevant to their desired career paths!

PTSD Prophylaxis & Suicidal Ideation (View on GitHub)
Examine a research cohort of medications, PTSD, and suicidality behavior to identify useful prophylatic medications for PTSD with adverse effects on suicidal ideation
Competencies: Data wrangling/cleaning; Classificiaton model fitting
High-Risk Patient Contact (View on GitHub)
Identify patients meeting age and BMI criteria for elevated complication risk and identify (simulated) patient contact information for follow-up
Competencies: Data wrangling/cleaning; Relational data/data merging; Joint case in Excel, Python, & R
VA & CMS Dual Enrollment (View on GitHub)
Identifying patients prescribed by both the veterans and medicare/medicaid health systems and visualizing overprescription of opioids
Competencies: Data wrangling/cleaning; Data visualization and presentation; Relational data/data merging
FAERS Data Aggregation (View on GitHub)
Extracting publicly hosted reports of adverse drug events from the FDA's available data and identifying a specific medication's reported adverse events
Competencies: Data access; Data wrangling/cleaning
